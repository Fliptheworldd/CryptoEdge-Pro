Quick FAQs

Q: Is my data safe?

Yes! All data is encrypted with AES-256 and stored locally. Nothing is sent to any cloud server.

Q: Do I need my own API keys?

Yes. You'll need Binance API keys (for trading) and OpenAI or Claude API keys (for AI analysis). We guide you through the setup.

Q: Can I lose money?

Crypto trading carries risk. Start with small amounts, use the Manual mode first, and always set stop-loss limits. Never invest more than you can afford to lose.

Q: How do updates work?

The app checks for updates automatically. When a new version is available, you'll see a notification and can download it with one click.
